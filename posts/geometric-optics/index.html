01 / 光的直线传播

影子与小孔成像

同一均匀介质中，光沿直线传播。

均匀介质 → 直线传播。

02 / 反射定律

入射角等于反射角

三线共面，两角相等。

1

三线共面

入射面内。

2

法线居中

分居两侧。

3

两角相等

θᵣ = θᵢ。

类型 表面 反射光 举例
镜面反射光滑平行反射平面镜、水面
漫反射粗糙散射各方向纸张、墙壁

💡 都遵守反射定律

漫反射让物体可见。

03 / 折射定律（斯涅尔定律）

n₁sinθ₁ = n₂sinθ₂

折射率越大，折射角越小。

📐 从光速比推导

折射率 n = c/v。

惠更斯原理推出

疏 → 密：偏向法线。

介质 折射率 n
真空1.000
空气1.000 3
水1.33
玻璃1.5 – 1.9
钻石2.42
θ₁=40° θ₂=25.4°

04 / 全反射

临界角与光纤

入射角 > 临界角 → 全反射。

1

光密 → 光疏

n₁ > n₂。

2

入射角 ≥ 临界角

θ₁ ≥ C 时全反射。

📐 临界角推导

令 θ₂ = 90°

C ≈ 41.8°。

纤芯 n 大，包层 n 小，全反射传光。

📡

高带宽

互联网骨干网。

💎

钻石闪耀

临界角小 → 火彩强。

05 / 增强练习

从基础到挑战

Q1

空气 → 水，θ₁=45°。

📖 Q1 答案

θ₂ ≈ 32.1°。

Q2

距镜 30 cm。

📖 Q2 答案

物像距离 60 cm，虚像。

Q3

水 → 空气。

📖 Q3 答案

C ≈ 48.8°。

Q4

玻璃 → 水，θ₁=30°。

📖 Q4 答案

可全反射，C ≈ 62.5°。

Q5

水下看水面。

📖 Q5 答案

半顶角 = 临界角 ≈ 48.8°。

Q6

平行玻璃板，θ₁=60°。

📖 Q6 答案

出射与入射平行，侧向平移。

01 / 三种力

重力、弹力与摩擦力

几乎所有受力分析题都从这三种力开始。

重力方向竖直向下，作用点在重心。

📐 公式解读

g = GM/R² ≈ 9.8 m/s²。赤道偏小，两极偏大。

重力 ≈ 万有引力（差异 < 0.3%）。

重心的位置

均匀物体重心在几何中心；不倒翁重心低所以不倒。

支持力、拉力本质都是弹力。

📐 公式解读

k 由弹簧材料决定；x 是形变量，不是总长。

仅在弹性限度内成立。

🔗 绳的拉力

沿绳方向，只能拉不能推。

📐 面的支持力

垂直接触面，N 不一定等于 mg。

阻碍接触面间相对运动（或趋势）的力。

比较项 静摩擦力 滑动摩擦力
产生条件有趋势，未滑动已发生相对滑动
大小随外力变化，最大为 f_maxf = μN，恒定
方向与运动趋势相反与相对运动相反
关键公式平衡/牛顿定律求解f = μN
📐 公式解读

μ 由接触面决定；N 是法向力，不是重力。

02 / 受力分析四步法

「一重二弹三摩擦」口诀

画错力，后面全错。

1

选研究对象

隔离法：从系统中取出单个物体。

2

画重力

竖直向下，从重心出发。

3

找弹力（接触力）

逐个检查接触面、绳索、弹簧。

4

查摩擦力 + 验证

合力方向必须与加速度一致。

常见陷阱

不要画'运动力'——必须有施力物体。

平行四边形法则与正交分解

力的等效替代——合成与分解是同一枚硬币的两面。

θ 越大，合力越小。

θ = 90° 时，F = √(F₁² + F₂²)。

调整角度，观察力的分解。

建系 → 投影 → 分轴列方程。

分解各力

04 / 平衡条件

共点力平衡：ΣF = 0

静止或匀速 → 合力为零。

📐 公式解读

正交分解是通用方法；二力/三力可用快捷法。

二力平衡

两力等大、反向、共线。

三力平衡

任意两力的合力与第三力等大反向。

正交分解法

多力平衡的万能解法。

05 / 增强练习

从基础到挑战

先独立思考，再展开答案。

B1. 重力计算

m = 3 kg，g = 10 m/s²。

📖 B1 答案

G = 30 N ↓

N = G = 30 N ↑

B2. 胡克定律

k = 200 N/m，拉伸到 20 cm。

📖 B2 答案

x = 0.05 m

F = 10 N

B3. 滑动摩擦力

5 kg，μ = 0.4，匀速。

📖 B3 答案

N = mg = 50 N

f = 20 N，F = 20 N

A1. 斜面平衡

m = 2 kg，θ = 30°。

📖 A1 答案

f = mg sin30° = 10 N ↗

N = mg cos30° ≈ 17.3 N

A2. 合力范围

F₁ = 5 N，F₂ = 3 N。

📖 A2 答案

2 N ≤ F ≤ 8 N

cosθ = −0.3，θ ≈ 107.5°

A3. 绳悬挂灯

OA 偏 30°，OB 水平。

📖 A3 答案

三力平衡于 O 点。

T₁ ≈ 46.2 N

T₂ ≈ 23.1 N

C1. 斜面 + 摩擦 + 推力

匀速上行，μ = 0.5，θ = 37°。

📖 C1 答案

N = 32 N

f = 16 N（沿斜面向下）

F = 40 N

C2. 三力平衡 — 矢量三角形

绳偏 60°，水平力 F。

📖 C2 答案

三力：G↓、T↗、F→

T = 20 N

F ≈ 17.3 N

力三角形法同样可解。

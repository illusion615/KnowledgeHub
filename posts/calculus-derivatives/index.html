01 / 从平均到瞬间

导数到底是什么？

上一篇我们知道了：把 Δx 缩到无穷小（取极限），就能从平均变化率跳到瞬时变化率。这个极限值，就叫做导数。

对于函数 f(x)，它在点 x 处的导数定义为：

翻译成人话：

1

取一个小增量 Δx

从 x 移动一小步到 x + Δx。

2

算函数值的变化

f(x + Δx) − f(x) 就是 y 值变了多少。

3

算平均变化率

变化量除以步长 = [f(x+Δx) − f(x)] / Δx。

4

让步长趋向零

取 Δx → 0 的极限，得到精确的瞬时变化率。

你车上的速度表显示的就是导数：

📏

里程表

记录的是你总共走了多远——这是函数 s(t)，距离随时间的函数。

⚡

速度表

显示的是你此刻有多快——这是 s'(t)，距离函数的导数，也就是速度 v(t)。

🎢

推背感

你感受到的加速/减速的力——这是 v'(t)，速度的导数，也就是加速度 a(t)。导数的导数！

02 / 动手试试

亲手算一个导数

来，我们用最简单的例子——f(x) = x²——走一遍求导过程。这是你真正"理解"导数的关键。

1

写出 f(x + Δx)

f(x + Δx) = (x + Δx)² = x² + 2x·Δx + (Δx)²

2

计算变化量 f(x + Δx) − f(x)

= (x² + 2x·Δx + (Δx)²) − x² = 2x·Δx + (Δx)²

3

除以 Δx

= (2x·Δx + (Δx)²) / Δx = 2x + Δx

4

取极限 Δx → 0

lim (2x + Δx) = 2x  （Δx → 0 时 Δx 项消失）

这意味着什么？

当 x = 3 时，f'(3) = 6，表示在 x = 3 这个位置，曲线 y = x² 的"陡峭程度"是 6——也就是说，x 每增加一个极小单位，y 大约增加 6 倍那么多。当 x = 0 时，f'(0) = 0——曲线在最低点是"平的"，完全不上升也不下降。

让我们在 x = 3 处，用越来越小的 Δx 来验证 f'(3) 确实等于 6：

Δx f(3 + Δx) = (3 + Δx)² [f(3+Δx) − f(3)] / Δx
116(16 − 9) / 1 = 7
0.19.61(9.61 − 9) / 0.1 = 6.1
0.019.0601(9.0601 − 9) / 0.01 = 6.01
0.0019.006001= 6.001
0.00019.00060001= 6.0001
→ 0→ 9→ 6 ✓

Δx 越小，结果越接近 6。这就是极限的力量——不需要 Δx 真的变成 0，只需要观察趋势就能得到精确答案。

03 / 求导捷径

四条让你事半功倍的法则

每次都用定义去推导太慢。数学家总结了一些规律（可以用定义严格证明），让求导变得像查字典一样快。

幂法则 Power Rule

"指数拿下来乘，原来的指数减 1。"

f(x)用幂法则f'(x)
x²2 · x²⁻¹2x
x³3 · x³⁻¹3x²
x⁵5 · x⁵⁻¹5x⁴
x¹ = x1 · x⁰1
x⁰ = 1（常数）0 · x⁻¹0

常数的导数为零

这很好理解——一个不变的量，它的"变化率"当然是 0。比如你停在原地不动，速度自然是 0。

常数倍法则

"常数可以直接提出来。" 比如 (3x²)' = 3 · 2x = 6x。

加法法则（和差法则）

"各部分分别求导再加起来。" 比如 (x³ + 5x²)' = 3x² + 10x。

有了这三条法则，你已经可以求任意多项式的导数了！试试这个：

导数还能告诉我们函数图像的"地形"——哪里上升、哪里下降、哪里是顶峰或谷底。

📈

f'(x) > 0

函数在上升——图形向右上方走。

📉

f'(x) < 0

函数在下降——图形向右下方走。

⛰️

f'(x) = 0

函数到了顶峰或谷底——刚好"拐弯"的地方。

这就是优化的基础

要找一个函数的最大值或最小值，只要找到 f'(x) = 0 的点——在工程中，这叫做优化问题。比如：用固定长度的铁丝围一个矩形牧场，怎么围面积最大？答案就藏在导数等于零的地方。

04 / 导数在行动

三个真实世界的案例

学了导数不是只为了做题——工程师、科学家和设计师每天都在用它。

过山车工程师设计轨道时，最关心的不是速度——而是加速度（速度的导数）和加加速度（加速度的导数，也叫 jerk）。

v

速度 v(t)：轨道的速率

过山车在最高点速度最慢（势能最大），最低点速度最快（动能最大）。

a

加速度 a(t) = v'(t)：乘客感受到的推力

a(t) 太大 → 乘客不舒服甚至危险。设计标准限制在 ±4g 以内。

j

jerk j(t) = a'(t)：推力变化的突然程度

如果加速度突然变化（jerk 很大），人会感到"被猛推"。好的过山车设计要让 jerk 尽量平滑。

一条过山车轨道就是一个空间曲线函数，工程师通过计算它的一阶、二阶和三阶导数来确保安全和舒适。

无人机的飞行控制器每秒计算数百次导数：

📍

位置传感器（GPS）

告诉飞控"我现在在哪"——这是 s(t)。

🧭

速度估算 = s'(t)

飞控对位置求导得到速度，判断是否偏离航线。

🔄

加速度 = s''(t)

再求一次导，得到加速度，用来调整电机输出。

当无人机遇到一阵大风被吹偏时，导数帮助飞控在几毫秒内检测到偏离，计算修正量并调整四个电机的转速——这一切都是导数的实时应用。
